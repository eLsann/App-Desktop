# INSTALLATION INSTRUCTIONS
# =============================================================================
# 1. Install Python 3.11.9 or later
#    Download: https://www.python.org/downloads/release/python-3119/
#
# 2. Create virtual environment:
#    python -m venv app.venv
#
# 3. Activate virtual environment:
#    Windows: app.venv\Scripts\activate
#    Linux/Mac: source app.venv/bin/activate
#
# 4. Install dependencies:
#    pip install -r requirements.txt
#
# 5. Install VLC Media Player:
#    Download: https://www.videolan.org/vlc/
#    Install to: C:\Program Files\VideoLAN\VLC
#
# 6. Configure environment:
#    copy .env.example .env
#    Edit .env with your Docker API settings
#
# 7. Run application:
#    python app.py (original features)
#    python app_enhanced.py (enhanced features)

# =============================================================================
# TROUBLESHOOTING GUIDE
# =============================================================================
# Issue: PySide6 import error
# Solution: pip install --upgrade PySide6
#
# Issue: MediaPipe TensorFlow conflict
# Solution: pip install --upgrade mediapipe
#
# Issue: VLC not found
# Solution: Install VLC to C:\Program Files\VideoLAN\VLC
#
# Issue: Camera not detected
# Solution: Check camera index in .env (CAM_INDEX=0,1,2...)
#
# Issue: Docker API connection failed
# Solution: Check Docker container: docker ps
